Quick Installation

Windows Setup

  1. Download the latest release zip.
  2. Extract and run run.bat.
  3. Press Ctrl + Home anywhere to open the bubble.
  4. Type your text and press Ctrl + End to inject!

macOS Setup

  1. Download the `.app` or zip file.
  2. Move it to your Applications folder.
  3. Grant Accessibility permissions (System Settings > Privacy > Accessibility) so it can type for you.
  4. Press Cmd + Home to open.

Linux Setup (Wayland & X11)

  1. Clone the repository or download the source.
  2. Install dependencies: sudo pacman -S wl-clipboard wtype (Arch)
  3. Set up a Custom Shortcut in your DE (e.g. Plasma Settings) pointing to run.sh.
  4. Assign it to Ctrl + Home.
