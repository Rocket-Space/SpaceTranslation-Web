Quick Installation

Windows Setup

Supported Systems: Windows 10, Windows 11

  1. Download the latest release zip.
  2. Extract and run the application. No additional dependencies required (Plug and Play).
  3. Press Ctrl + Home anywhere to open the bubble.
  4. Type your text and press Ctrl + End to inject!

macOS Setup

Supported Systems: macOS 10.15 (Catalina) or higher

  1. Download the `.zip` file and extract the app.
  2. Move it to your Applications folder.
  3. Crucial: Grant Accessibility permissions (System Settings > Privacy & Security > Accessibility) so it can type for you.
  4. Press Cmd + Home to open.

Linux Setup (Wayland & X11)

Supported Systems: Any modern distribution (Ubuntu, Fedora, Arch, Mint, etc.)

  1. Download the `.tar.gz` and extract it.
  2. Dependencies: Install the virtual keyboard tools for text injection:
    • Wayland: sudo apt install wl-clipboard wtype
    • X11: sudo apt install xclip xdotool
  3. Set up a Custom Shortcut in your DE (e.g. Settings > Keyboard > Shortcuts) pointing to the binary.
  4. Assign it to Ctrl + Home.
